# 🛒 India-Cart

**India-Cart** is a full-stack E-commerce web application that simulates an online shopping platform.  
It allows users to browse products, manage cart items, and place orders using a responsive frontend and a Spring Boot backend with PostgreSQL database.

---

## 🚀 Features

- 🛍️ Product listing with categories
- 🔍 Search and filter products
- 🛒 Add to Cart / Remove from Cart
- 📦 Order placement system
- 🔁 RESTful API integration
- 🧱 Layered architecture (Controller → Service → Repository)
